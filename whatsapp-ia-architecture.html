Stack Tecnológico

▼
API Gateway
WhatsApp via Meta
Cloud API (Meta) + webhook HTTPS. En local se usa ngrok o cloudflared para exponer el puerto.
Webhook Service
FastAPI / Node.js
Valida firma HMAC-SHA256 de Meta, encola mensajes en Redis y responde 200 OK en <2s.
IA Core
Python · FastAPI
Orquesta LLM, RAG y herramientas MCP. LangChain / LlamaIndex para el pipeline RAG.
LLM
OpenAI / Ollama
Dev local: Ollama con llama3.2 o mistral. Prod: GPT-4o / Claude Sonnet.
Cache
Redis
Historial de conversación en memoria (TTL 24h), rate-limiting por número, cola de tareas con BullMQ.
Base de datos
PostgreSQL 16
Usuarios, historial largo, configuraciones, logs de herramientas.
Vector DB
Qdrant
Embeddings de documentos para búsqueda semántica (RAG). text-embedding-3-small o nomic-embed.
MCP Protocol
MCP Servers
Herramientas extensibles via stdio/SSE: filesystem, postgres, brave-search, y servidores custom.

Integración MCP — Model Context Protocol

▼

El MCP Client Hub en ia-core se conecta a cada servidor MCP y expone sus herramientas al LLM como funciones.

# services/ia-core/app/mcp/client.py
from mcp import ClientSession, StdioServerParameters
from mcp.client.stdio import stdio_client

class MCPHub:
    servers = {
        "filesystem": StdioServerParameters(
            command="npx",
            args=["-y", "@modelcontextprotocol/server-filesystem", "/app/docs"]
        ),
        "postgres": StdioServerParameters(
            command="npx",
            args=["-y", "@modelcontextprotocol/server-postgres"],
            env={"DATABASE_URL": "${DATABASE_URL}"}
        ),
        "custom": StdioServerParameters(
            command="python",
            args=["/app/mcp-servers/custom/server.py"]
        ),
    }

    async def get_tools(self, server_name: str) -> list:
        async with stdio_client(self.servers[server_name]) as (r, w):
            async with ClientSession(r, w) as session:
                await session.initialize()
                tools = await session.list_tools()
                return tools.tools

    async def call_tool(self, server: str, tool: str, args: dict):
        async with stdio_client(self.servers[server]) as (r, w):
            async with ClientSession(r, w) as session:
                await session.initialize()
                result = await session.call_tool(tool, args)
                return result
⚡ Para agregar un nuevo MCP server: crea el archivo en mcp-servers/custom/server.py, regístralo en MCPHub.servers y reinicia ia-core. Sin cambios en el LLM ni en el webhook.
⚡

Flujo de Mensaje (Request → Response)

▼
  1. Usuario envía mensaje en WhatsApp → Meta llama al webhook HTTPS en tu servidor (ngrok en local)
  2. Webhook service valida la firma SHA256 de Meta, extrae el texto/media y publica en cola Redis con BullMQ
  3. ia-core worker consume el mensaje de la cola, recupera contexto de conversación de Redis (últimos N turnos)
  4. RAG Engine genera embedding del mensaje y busca chunks relevantes en Qdrant (si aplica)
  5. Orchestrator construye el prompt con: system prompt + contexto Redis + chunks RAG + herramientas MCP disponibles
  6. LLM call (OpenAI / Ollama): si el modelo invoca una tool, el orchestrator la ejecuta via MCP Hub y vuelve a llamar al LLM con el resultado
  7. Respuesta final → ia-core llama a la API de WhatsApp (Meta Graph) para enviar el mensaje de vuelta al usuario
  8. Persistencia: el turno completo se guarda en PostgreSQL para historial largo y auditoría

Setup Local — Paso a Paso

▼
#PasoComando / Acción
01 Clonar y configurar env cp .env.example .env → llenar tokens
02 Levantar infraestructura docker compose up -d redis postgres qdrant
03 Ejecutar migraciones docker compose run ia-core python -m alembic upgrade head
04 Crear colección Qdrant docker compose run ia-core python scripts/init_qdrant.py
05 Ingestar documentos docker compose run ia-core python ingestion/ingest_pdf.py ./docs/
06 Levantar LLM local (opcional) docker compose up -d ollama → ollama pull llama3.2
07 Levantar todos los servicios docker compose up -d
08 Exponer con ngrok docker compose up ngrok → copiar URL HTTPS
09 Registrar webhook en Meta Meta Developers → {ngrok_url}/webhook → verify token
10 Test de extremo a extremo Envía "hola" desde WhatsApp 🎉
